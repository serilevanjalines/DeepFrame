# DeepFrame - Documentary Streaming Platform

A React-based web application showcasing a curated collection of documentaries with search and discovery features.

## Features
- **Search Functionality**: Real-time search across documentary titles and descriptions
- **Responsive Card Layout**: Clean, Netflix-inspired card grid design
- **Dynamic Content**: 6+ documentaries with images, descriptions, and metadata
- **Category Organization**: Filter by documentary categories
- **Fast Navigation**: Smooth user experience with React components

## Tech Stack
- **Frontend**: React (JavaScript JSX)
- **Styling**: CSS with modern responsive design
- **Backend**: Node.js with Express server
- **Data**: Structured JSON card data for easy content management

## Documentary Catalog Includes
- The Case for Christ
- Unacknowledged (UFO Documentary)
- The Secret of Skinwalker Ranch
- Secrets of the Saqqara Tomb
- The Social Dilemma
- 13th (Criminal Justice Documentary)
